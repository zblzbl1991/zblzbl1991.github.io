CMD :用于指定默认的容器主进程的启动命令的
ENTRYPOINT: 用于指定容器运行程序和参数

  1. redis
    1. 首先安装下 docker pull redis:latest
    2. 然后运行redis容器 docker run -itd --name redis-test -p 6379:6379 redis 参数的作用:
    • -i 以交互模式运行容器，通常与 -t 同时使用；
    • -t 为容器重新分配一个伪输入终端，通常与 -i 同时使用；
    • -d 后台运行容器，并返回容器ID；
